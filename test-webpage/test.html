Now, let's get a table in here by using the table tag. This one does require a closing tag (table/). A table tag requires two other tags inside it: thead and tbody. Inside thead, you add the correct amount of th tags: their are the headers for the table (vertical sections). Inside tbody, you add the correct amount of tr tags, which stand for table row. Inside table row tags, you add the correct amount of td tags, table data. Take a look in the code for a better understanding. Let's see how all this looks like
Also notice that text inside a table tag can only be added through the use of other tags. For example (using [] instead of <>), "wasd" in [table]wasd[table/] would be displayed outside of the table
